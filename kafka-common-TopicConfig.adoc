== [[TopicConfig]] TopicConfig

`TopicConfig` are the <<keys, keys>> for a per-topic configuration.

[source, scala]
----
import org.apache.kafka.common.config.TopicConfig
assert(TopicConfig.CLEANUP_POLICY_CONFIG == "cleanup.policy")
----
